# ♻️ Afvalherkenner ♻️
Een webapp die afval herkent via de camera en automatisch het type afval bepaalt.

---

## Werking

1. Houd het afval dicht voor de camera  
2. Wacht ±3 seconden tot de scan voltooid is  
3. Er verschijnt een popup met het resultaat  
4. Klik op **Opnieuw scannen** om een nieuwe scan te starten  

## Opmerking

Zorg ervoor dat het afval goed zichtbaar en dicht bij de camera wordt gehouden voor de beste resultaten.
